## 📖 Overview

Budget Tracker is a personal finance web application that helps users manage their budget and expenses. Users can set a budget, record daily expenses, and monitor their remaining balance.

The application stores expense data locally using the browser's Local Storage.

---

## ✨ Features

- 💵 Set a budget
- ➕ Add expenses
- 📂 Categorize expenses
- 📅 Record expense date
- 🗑 Delete expenses
- 📊 View remaining balance
- 💾 Local Storage support

---

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Local Storage API
